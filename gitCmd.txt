Git commandes pour modifier des fichiers : 
1. From the project respo 
2. git pull origin main 
3. ..work locally.. 
4. git add <file_name> 
NB: replace the <file_name> by the exact name of the file 
5. git commit -m "your_message" (((IMPORTANT, toujours envoyer un message, car comme ca on  est sur que c'est une nouvelle version))) 
NB: replace "your_message" by "blabla" 
6. git push
